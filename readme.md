# 信息上报协议

信息上报内容包括输出电压，电流，输出功率，输出状态（on/off），系统时间(s)，输入电压，每四位16进制数对应一个数据

输出电流：00 00 00 00

输出功率：00 00 00 00

开关状态：00 00 00 01

系统时间：00 00 00 62

输入电压：00 00 23 0B
